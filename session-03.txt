
- Consider OpenShift cluster (3 master nodes , N worker nodes).
- HOw to Login to OCP cluster ?
  1- Token based authentication 
  2- Certificate based authentication.
  
1-# Token based authentication  
   User Name - admin 
   Password  - redhatocp 
   API URL   - https://api.ocp4.example.com:6443 
               https://api.clusterName.baseDomain:6443 
			   
   
   Login to OCP cluster:-   
   $ oc login -u admin -p redhatocp https://api.ocp4.example.com:6443 
   or
   Token; API URL 
   $ oc login --token <token> --server https://api.ocp4.example.com:6443 
   
   
   - TOken based authentication via IdP (Identity Provider).

	 LDAP
